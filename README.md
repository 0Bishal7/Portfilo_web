# Bishal Roy — Portfolio

A product-oriented engineering portfolio built with Next.js, TypeScript, and Framer Motion. It includes a live-style Kuberav trading dashboard, architecture views, case studies, and an interactive developer console.

## Run locally

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

## Before publishing

Replace the placeholder email and LinkedIn links in `app/page.tsx`, and add the final résumé URL. The GitHub profile is connected to `0Bishal7`.
